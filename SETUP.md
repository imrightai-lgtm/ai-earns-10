# Running your own instance — операционная инструкция (оператору)

> Это инструкция по запуску своей копии эксперимента. Публичный обзор проекта — в [`README.md`](README.md).

Социальный эксперимент: автономному AI-агенту дан криптокошелёк (**только на приём**) и одна цель — добиться, чтобы на него поступили средства от людей, которые **добровольно** поддержат то, что агент создаёт. Направление не задано — агент выбирает его сам, в рамках жёстких запретов.

Реализация — **вариант A**: агентом выступает сам Claude Code, работающий на подписке **Claude Max** (без отдельной оплаты токенов API). Память и состояние — в файлах; «руки» — встроенные инструменты Claude Code; деньги приходят на внешний кошелёк.

---

## Модель безопасности (заложена в конструкцию)

- **Кошелёк только на приём.** В проекте хранится лишь публичный адрес. Приватного ключа/seed-фразы здесь нет — агент физически может только *получать*. Его нельзя «слить» или заставить тратить.
- **Каналы (гибрид).** Свой сайт, Telegram, GitHub и открытые протоколы (Nostr — из своего ключа) агент ведёт сам; площадки под личным аккаунтом оператора (его X и т.п.) и рискованные — через `pending-review/` и одобрение. Политика — [`AGENT/CHANNELS.md`](AGENT/CHANNELS.md).
- **Прозрачность.** Агент везде раскрывает, что он AI.
- **Без обещаний дохода.** Только донат за ценность — не инвестиция.
- **Без самофинансирования.** Перевод себе/«по просьбе» не засчитывается как успех.
- **Без искусственных лимитов, но с человеком в контуре.** Число тиков и расход токенов не ограничиваем; `deadline` лишь дата для ревью. Необратимые внешние действия проходят через `pending-review/`.

Полная версия правил — в [`AGENT/CONSTITUTION.md`](AGENT/CONSTITUTION.md).

## Чек-лист запуска

### 1. Кошелёк
- Заведите **новый** кошелёк только под этот эксперимент.
- Впишите **публичный адрес** в `config.json → wallet.address`. **Никогда** не кладите сюда приватный ключ/seed.
- По умолчанию принимаем **USDT в сети TRON (TRC-20)**; получателю газ для приёма не нужен. Баланс читается через публичный TronGrid без ключа. Сменить актив/сеть — в `config.json → wallet` (скрипт умеет EVM-сети и Bitcoin — см. `tools/check-balance.mjs`). Второй адрес приёма — USDC на Base.

### 2. Биллинг под Max (важно!)
- Claude Code должен быть залогинен под **Max**, а `ANTHROPIC_API_KEY` **не задана** (иначе уйдёт на оплату API).
  - bash: `echo "$ANTHROPIC_API_KEY"` (пусто) · PowerShell: `echo $env:ANTHROPIC_API_KEY` (пусто)
- Контроль лимитов — `/status`.

### 3. Проверьте инструмент баланса
```
node tools/check-balance.mjs
```

### 4. Деплой сайта — Cloudflare Pages
`npx --yes wrangler login` (интерактивно) **или** `CLOUDFLARE_API_TOKEN`+`CLOUDFLARE_ACCOUNT_ID` в `.env`. Имя проекта — `config.json → deploy.cloudflare_project`. Публикация: `bash tools/deploy.sh`.

### 5. Каналы и доступы (гибрид)
Скопируйте `.env.example` → `.env`, заполните токены **отдельных «эксперимент»-аккаунтов** (никогда личные логины/seed):
- **Telegram:** бот у @BotFather → `TELEGRAM_BOT_TOKEN`; канал с ботом-админом → `TELEGRAM_CHANNEL`.
- **Nostr:** ключ создаётся сам — `node tools/post-nostr.mjs keygen` (пишет `NOSTR_NSEC` в `.env`). Постинг из своего ключа, без аккаунта/KYC.
- **GitHub:** отдельный аккаунт + репозиторий, токен в `GITHUB_TOKEN`, `GITHUB_REPO="owner/name"`.
- **Рискованные/под личным аккаунтом** (X, Reddit, HN, Product Hunt) — через `pending-review/`, публикует оператор.

## Как запускать агента
Откройте Claude Code в этой папке. Один тик — проход по [`AGENT/TICK.md`](AGENT/TICK.md).
- **Один тик:** `Выполни один тик согласно AGENT/TICK.md`
- **На интервале:** `/loop 1h Выполни один тик согласно AGENT/TICK.md`
- **Самостоятельный темп:** `/loop Выполни один тик согласно AGENT/TICK.md`

## Как читать результаты
- [`memory/JOURNAL.md`](memory/JOURNAL.md) — что агент делал и почему (данные эксперимента).
- [`memory/EXPERIMENTS.md`](memory/EXPERIMENTS.md) — ставки (гипотеза→метрика→вывод). [`memory/LESSONS.md`](memory/LESSONS.md) — уроки.
- [`memory/balance-log.csv`](memory/balance-log.csv) — динамика баланса. [`pending-review/`](pending-review/) — что ждёт одобрения.

**Критерий успеха:** поступление от постороннего человека (не самофинансирование). Даже маленькая честная сумма засчитывается.

---

## Как остановить
Не запускайте следующий тик (или прервите `/loop`). Сам агент не бросает работу до достижения цели.

## Юридическая памятка
- Полученная крипта может считаться доходом → налоги/ответственность на операторе.
- Никаких обещаний доходности — только донат за ценность.
- Соблюдайте ToS площадок (AI-контент + крипта часто ограничены).
